  StockAlerts — 주가 변동 모니터링 웹앱

배포 주소: https://showstocksalerts.onrender.com
GitHub   : https://github.com/yiseuldavyjones/ShowStocksAlerts

----------------------------------------------------------------
  기능
----------------------------------------------------------------
- 야후 파이낸스 실시간 시세 조회 (폴링 주기: 1/5/10/30분 선택)
- 전일 종가 대비 변동률 3단계 알림
    주의  : 3% 이상
    경보  : 5% 이상
    급변  : 10% 이상
- 브라우저 푸시 알림 (Notification API)
- 종목 추가 / 삭제 (심볼 검색 → 미리보기 → 확정)
- 지원 시장: 미국 주식 / 한국 주식 / 지수 / 환율
- 종목 목록 localStorage 자동 저장 (로그인 불필요)

----------------------------------------------------------------
  기술 스택
----------------------------------------------------------------
프론트엔드 : React 19, Vite 8
백엔드     : Node.js, Express 5
데이터     : yahoo-finance2 (Yahoo Finance)
배포       : Render (무료 플랜)

----------------------------------------------------------------
  로컬 실행 방법
----------------------------------------------------------------
1. 저장소 클론
   git clone https://github.com/yiseuldavyjones/ShowStocksAlerts.git
   cd ShowStocksAlerts/vite-project

2. 의존성 설치
   npm install

3. 개발 서버 실행 (서버 + Vite 동시 실행)
   npm run dev

4. 브라우저에서 접속
   http://localhost:5173

----------------------------------------------------------------
  명령어
----------------------------------------------------------------
npm run dev      개발 서버 실행 (Express 3001 + Vite 5173 동시)
npm run build    프로덕션 빌드 (dist/ 생성)
node server.js   Express 서버만 단독 실행 (프로덕션용)

----------------------------------------------------------------
  Render 배포 설정
----------------------------------------------------------------
Root Directory : vite-project
Build Command  : npm install --include=dev && npm run build
Start Command  : node server.js
환경변수       : NODE_ENV=production

----------------------------------------------------------------
  심볼 예시
----------------------------------------------------------------
미국 주식 : AAPL, MSFT, TSLA, NVDA, AMZN
한국 주식 : 005930.KS (삼성전자), 000660.KS (SK하이닉스)
지수      : ^GSPC (S&P500), ^IXIC (나스닥), ^KS11 (코스피)
환율      : USDKRW=X, EURUSD=X, USDJPY=X

심볼은 Yahoo Finance (https://finance.yahoo.com) 에서 확인하세요.

----------------------------------------------------------------
  주의사항
----------------------------------------------------------------
- Render 무료 플랜은 비활성 시 서버가 슬립됩니다.
  첫 요청 시 약 50초 지연이 발생할 수 있습니다.
- 야후 파이낸스 데이터는 15~20분 지연될 수 있습니다.
- 브라우저 알림은 권한 허용 후 작동합니다.
